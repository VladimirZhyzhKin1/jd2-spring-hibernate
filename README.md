# News management
## Description
Web project for posting / viewing news.</br>
News manager updated project using technology with Spring, hibernate

Web application is built on MVC principle of using Layered Architecture.

## Technologies
### Used Technologies:
- Java 8
- Spring MVC
- Spring IoC
- Spring Security
- Spring AOP
- Hibernate
- MySQL
- JSP/JSTL/CSS3
- log4j
